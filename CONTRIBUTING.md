# Contributing

Thanks for your interest in Cyber Katalog. These conventions apply across our repositories.

## Workflow

- Branch from `master` using `auto/<topic>`; never commit directly to `master`.
- Open a pull request for every change. Keep commit and PR titles in English, following Conventional Commits (`feat:`, `fix:`, `docs:`, `ci:`, ...).
- Make sure the repository checks pass before requesting review.

## Style

- Use a plain hyphen `-` in text; never an en-dash or em-dash.
- No AI-generated attribution in commits, pull requests or docs.

## Security

Please report vulnerabilities privately - see [SECURITY.md](./SECURITY.md).
